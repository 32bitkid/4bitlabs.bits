Function createBitReader

  • createBitReader(source, options?): BitReader

  • Create a new BitReader from source bytes.

    Parameters

    • source: ArrayBuffer | TypedArray

      The source data for the reader.

    • options: BitReaderOptions = {}

    Returns BitReader

    Example: Reading in most-significant bits first mode.

    const source = Uint8Array.of(0b1110_0001);
    const br = createBitReader(source, { mode: 'msb' });

    br.read32(3); // 0b111
    br.read32(1); // 0b0
    br.read32(3); // 0b000
    br.read32(1); // 0b1

    Example: Reading in least-significant bits first mode.

    const source = Uint8Array.of(0b1110_0001);
    const r = createBitReader(source, { mode: 'lsb' });

    br.read32(3); // 0b001
    br.read32(1); // 0b0
    br.read32(3); // 0b110
    br.read32(1); // 0b1
